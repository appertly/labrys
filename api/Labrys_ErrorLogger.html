Creates a new ErrorLogger.

If an exception isn't found in the `$levels` map, this class assumes a

level of `LogLevel::ERROR`.

```hack

$elog = new ErrorLogger(

$logger,

Map {"MyException" => LogLevel::DEBUG, "RuntimeException" => LogLevel::WARN}

);

```
