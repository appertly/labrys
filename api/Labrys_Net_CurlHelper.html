exec

protected function exec(mixed $urlOrHandle)

A convenience wrapper around[`curl_multi_await`](/hack/reference/function/curl_multi_await/).

Pass a cURL handle, or, more simply, a string containing a URL (and the

cURL handle will be created for you), and the cURL request will be executed

via async and the `string` result will be retuned.

Parameters
NameTypeDescription
$urlOrHandleresource | string- An existing cURL handle or a `string` URL. String URLs will create a default cURL GET handle.
Returns

Throws
