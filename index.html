Labrys

A library that helps you write web applications in Hack and HHVM

Install via Composer

composer install appertly/labrys

Contribute

Find the project on GitHub where you can submit issues or pull requests.

Documentation

A reference guide is in the works. In the meantime, check out the README or take a look at the API documentation.

The Gist

Really, Labrys is the glue between several micro libraries.

Features

  • Using the caridea/container library, we declare three dependency containers: one for configuration properties, one for back-end objects (like services), and one for front-end objects (like controllers).
    • You can register custom modules with the system. These modules can register objects in these dependency containers.
  • We provide several web utility classes. This includes a Front Controller that accepts PSR-7 Request and Response objects, and using the aura/router library, dispatches requests to a controller that you specify.
    • We provide two helper traits, one for controllers to use, and one to use when generating REST JSON responses.
    • The Front Controller includes a Contingency object which handles errors.
    • We provide interfaces and XHP that allows your modules to include blocks of content in predefined regions of page content.
  • You'll find some helper classes for working with databases. Right now, we include MongoDB helpers.
  • We have classes that expose the output from system processes as well as MongoDB GridFS files as Psr\Http\Message\StreamInterface objects.
  • We have a helper trait for working with cURL — it actually throws good exceptions.
